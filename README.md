# React Portfolio

Portfolio page built with React + Vite using details from your resume.

## Includes

- Resume-based profile summary and highlights
- GitHub and LinkedIn navigation links
- Dynamic styling controls (palette, accent color, corner radius, spacing density)
- Responsive layout for desktop and mobile

## Run

```bash
npm install
npm run dev
```

## Update social links

Edit `src/profileData.js`:

- `links.github`
- `links.linkedin`
